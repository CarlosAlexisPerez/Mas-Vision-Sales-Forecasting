# Imágenes del Proyecto

Esta carpeta contiene las principales visualizaciones, dashboards y gráficos utilizados durante el desarrollo del proyecto de análisis y predicción de ventas para Más Visión.

Las imágenes permiten ilustrar los hallazgos obtenidos mediante análisis exploratorio, visualización comercial y modelado predictivo.

---

## Contenido

- Dashboard ejecutivo en Power BI
- Forecast de ventas 2025
- Resultados del modelo ARIMA

---

## Objetivo

Facilitar la interpretación visual de los resultados y complementar la documentación técnica del proyecto.
