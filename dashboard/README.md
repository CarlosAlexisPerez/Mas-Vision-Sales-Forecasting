# Dashboard Power BI

Esta carpeta contiene el dashboard ejecutivo desarrollado en Power BI para el análisis comercial y monitoreo de ventas históricas de Más Visión.

El dashboard fue diseñado con el objetivo de facilitar la toma de decisiones mediante visualizaciones dinámicas, indicadores clave y análisis interactivo del desempeño comercial.

---

## Funcionalidades principales

- Seguimiento de ventas totales
- KPIs comerciales
- Tendencias históricas de ventas
- Comparativa por formato de tienda
- Distribución por categorías de producto
- Análisis de descuentos
- Filtros dinámicos por año y mes
- Visualización interactiva de insights comerciales

---

## Herramientas utilizadas

- Power BI
- Power Query
- DAX
- Python
- Excel

---

## Objetivo

Proporcionar una herramienta visual e interactiva para analizar el comportamiento histórico de ventas y apoyar la toma de decisiones estratégicas basadas en datos.

---

## Insights principales

- Crecimiento sostenido de ventas entre 2022 y 2024
- Mayor participación comercial del formato E
- Impacto positivo de descuentos en determinados periodos
- Identificación de patrones estacionales
- Diferencias de desempeño entre sucursales
