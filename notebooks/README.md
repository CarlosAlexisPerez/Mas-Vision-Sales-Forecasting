# Notebooks

Esta carpeta contiene los notebooks desarrollados durante las distintas etapas del proyecto de análisis y predicción de ventas para Más Visión.

Los notebooks incluyen procesos de limpieza de datos, análisis exploratorio, visualización, modelado predictivo y forecasting de ventas mediante series temporales.

---

## Contenido

### 01_EDA_MasVision.ipynb
Notebook enfocado en:

- Limpieza y validación de datos
- Análisis Exploratorio de Datos (EDA)
- Visualización de tendencias comerciales
- Evaluación de estacionalidad
- Impacto de descuentos
- Análisis por categorías y formatos de sucursal

### 02_ARIMA_Sales_Forecasting.ipynb
Notebook enfocado en:

- Preparación de series temporales
- Entrenamiento del modelo ARIMA
- Evaluación de métricas
- Forecasting de ventas
- Predicción comercial para 2025
- Comparación entre valores reales y predicciones

---

## Herramientas utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Statsmodels

---

## Objetivo

Documentar el flujo completo de análisis y modelado predictivo utilizado para generar insights comerciales y proyecciones estratégicas de ventas.
