# Presentación Ejecutiva

Esta carpeta contiene la presentación ejecutiva desarrollada para comunicar los resultados obtenidos durante el proyecto de análisis y predicción de ventas para Más Visión.

La presentación resume el proceso completo del proyecto, incluyendo:

- Objetivos de negocio
- Metodología aplicada
- Análisis Exploratorio de Datos (EDA)
- Dashboard comercial en Power BI
- Modelo predictivo ARIMA
- Forecasting de ventas 2025
- Insights estratégicos
- Recomendaciones comerciales

---

## Objetivo

Comunicar de manera clara y ejecutiva los hallazgos, resultados y valor generado mediante técnicas de Ciencia de Datos y Business Intelligence.

---

## Contenido principal

- Contexto del problema de negocio
- Análisis histórico de ventas
- Evaluación de tendencias comerciales
- Impacto de descuentos
- Predicciones de ventas
- Métricas del modelo
- Recomendaciones estratégicas
- Conclusiones del proyecto

---

## Herramientas utilizadas

- PowerPoint
- Python
- Power BI
- Excel
