# Predicciones

Esta carpeta contiene las predicciones de ventas generadas para el año 2025 mediante modelos de series temporales.

Las proyecciones fueron desarrolladas utilizando técnicas de Machine Learning y forecasting orientadas a apoyar la planeación estratégica y comercial de la empresa.

## Contenido
- Predicciones mensuales de ventas
- Resultados del modelo ARIMA
- Forecast comercial 2025

## Objetivo
Proporcionar estimaciones de ventas futuras para apoyar la toma de decisiones basada en datos y mejorar la planificación comercial.
