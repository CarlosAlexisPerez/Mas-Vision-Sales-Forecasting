# Datos Originales

Esta carpeta contiene el dataset original proporcionado para el desarrollo del proyecto.

El archivo representa la información histórica de ventas de Más Visión correspondiente al periodo 2022–2024, incluyendo variables comerciales y categorías de productos utilizadas para el análisis y modelado predictivo.

## Contenido
- Datos originales sin procesamiento
- Información histórica de ventas
- Variables comerciales por sucursal

## Objetivo
Conservar una copia íntegra de los datos fuente utilizados durante el proyecto de análisis y predicción de ventas.
