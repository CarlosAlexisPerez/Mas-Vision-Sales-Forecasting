Esta carpeta contiene los datasets utilizados durante el desarrollo del proyecto de predicción de ventas para Más Visión.

La información incluye datos históricos de ventas, datasets procesados para análisis y modelado, así como las predicciones generadas para el año 2025.

## Estructura

### raw/
Contiene los datos originales proporcionados para el proyecto.

- `masvision_sales_original.xlsx`

### processed/
Contiene los datasets limpios, transformados y enriquecidos utilizados para el análisis exploratorio y modelado predictivo.

- `masvision_sales_limpio.csv`

### predictions/
Contiene las predicciones generadas por el modelo de Machine Learning para el año 2025.

- `sales_forecast_2025.csv`

## Tecnologías utilizadas
- Python
- Pandas
- NumPy
- Excel

## Objetivo
Centralizar y organizar los datos utilizados en las distintas etapas del proyecto:
- Limpieza de datos
- Análisis Exploratorio (EDA)
- Visualización en Power BI
- Modelado predictivo
- Forecasting de ventas
