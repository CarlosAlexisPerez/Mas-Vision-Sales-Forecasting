# Datos Procesados

Esta carpeta contiene el dataset limpio, transformado y enriquecido utilizado durante el análisis exploratorio de datos (EDA), visualización y modelado predictivo.

Los datos fueron preparados mediante procesos de limpieza, validación y transformación para garantizar consistencia y calidad en el análisis.

## Procesos realizados
- Limpieza de datos
- Validación de valores nulos
- Corrección de formatos
- Transformación de variables
- Integración de descuentos comerciales
- Preparación para modelado predictivo

## Objetivo
Facilitar el análisis de datos y mejorar el desempeño del modelo de predicción de ventas.
